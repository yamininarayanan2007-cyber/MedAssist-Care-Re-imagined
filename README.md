
# MedAssist-Care,Reimagined-AI Symptom Triage Chatbot

## Overview

The **AI Symptom Triage Chatbot** is a healthcare support system designed to assist health workers in analyzing patient symptoms and providing preliminary health guidance. The chatbot helps in identifying possible health conditions based on symptoms entered by the user and provides basic medical advice.

This system is intended to support **early health assessment**, especially in areas where immediate medical consultation may not be available.

---

## Key Features

* Patient symptom input through a chatbot interface
* AI-based symptom analysis
* Risk level indication (Low / Medium / High)
* Health awareness and prevention suggestions
* Nearest hospital recommendation
* Simple and user-friendly interface

---

## System Workflow

1. Health worker enters patient symptoms in the chatbot interface.
2. The frontend sends the information to the backend system.
3. The AI component analyzes the symptoms.
4. The system determines the possible health risk level.
5. The chatbot provides health advice and hospital suggestions.
